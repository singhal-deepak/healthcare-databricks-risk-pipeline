# Healthcare Readmission Risk Pipeline (Databricks)

## Project Overview
This project builds an end-to-end healthcare data pipeline using Databricks and Delta Lake to predict patient readmission risk.

## Architecture
Bronze → Raw Patient Data  
Silver → Cleaned & Standardized Data  
Gold → Patient Risk Features  
ML → Readmission Prediction Model  

## Tech Stack
- Databricks Community Edition
- PySpark
- Delta Lake
- Python
